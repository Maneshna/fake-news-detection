Fake News Detection Project

This website is designed to identify whether a news article is real or fake using a machine learning based model.

Objective

To detect misleading or false news content using text analysis and prediction.

Technology Used

HTML, CSS, JavaScript, Flask, Python, Pytorch, Scikit-learn, nltk, transformers.

Model Accuracy

The model gives an approximate prediction accuracy of 92%.
